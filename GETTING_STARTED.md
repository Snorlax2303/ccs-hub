# 🚀 COMECE AQUI — Guia de Primeiros Passos

Bem-vindo ao CCS Hub! Este guia te leva pelos primeiros passos para colocar o sistema rodando.

⏱️ **Tempo total**: ~1 hora

---

## ✅ Checklist de Setup

### Fase 1: Infraestrutura (15 min)

- [ ] **1. Supabase**
  - [ ] Criar conta em [supabase.com](https://supabase.com)
  - [ ] Ler `SETUP_SUPABASE.md`
  - [ ] Importar schema SQL
  - [ ] Copiar credenciais para `.env`

- [ ] **2. DeepSeek**
  - [ ] Ler `SETUP_DEEPSEEK.md`
  - [ ] Confirmar que API key está em `.env`

### Fase 2: Backend (20 min)

\`\`\`bash
cd backend
npm install
npm run dev
\`\`\`

✅ Você deve ver: \`Backend rodando em http://localhost:3001\`

### Fase 3: Frontend (20 min)

\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`

✅ Você deve ver: \`Frontend rodando em http://localhost:3000\`
