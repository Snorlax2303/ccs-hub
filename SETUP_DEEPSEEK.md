# 🤖 Setup DeepSeek — API de IA

## ✅ Você Já Tem!

Sua API key está configurada:

```
DEEPSEEK_API_KEY=sk-your-api-key-here
```

## 1️⃣ Verificar Crédito

1. Acesse [platform.deepseek.com](https://platform.deepseek.com)
2. Faça login
3. Vá para Billing
4. Veja quanto crédito você tem

## 2️⃣ Configurar no Projeto

A key já está no .env.example:

```env
DEEPSEEK_API_KEY=sk-your-api-key-here
DEEPSEEK_MODEL=deepseek-chat
```

## 3️⃣ Casos de Uso

DeepSeek será usado para:
1. Análise de Contratos
2. Alertas Inteligentes
3. Extração de PDFs
4. Geração de Templates

## 💰 Custo

- Análise de 1000 contratos/mês: ~$0.02-0.05
- Com seus créditos: Meses de uso gratuito

## ✅ Pronto!

DeepSeek está configurado. Próximo passo: backend/frontend.
